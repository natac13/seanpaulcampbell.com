Challenges

#1 - Candies

The Problem

Given children (number) and candy (number) find out how many candies can be eaten in total if each child get equal amounts.

Solution

1const getTotalEaten = (children: number, candy: number) => {
2  return Math.floor(candy / children) * children
3}
4// alternative solution that I like much better
5const getTotalEatenImproved = (children: number, candy: number) => {
6  return candy - (candy % children)
7}

The first solution was my initial thought and from looking at other solutions, seemed to be the common result. This also uses the hint which was given in the video of Math.floor(). Where I divided candy by children to find the number each child would eat; then multiple it by the number of children to find total eaten.

The second solution was from another workers elf's tweet.

The use of Math.floor() is not needed with this method. Plus I find it much easier to reason about. Where we find the remaining number of candies after dividing candy by children and subtract that value from the total candies. This show the use of the % modulo operator.

#2 - Deposit Profit

Problem

Given a deposit, rate and threshold find the year when your deposit with interest is over the threshold.

Solution

1const depositProfit(deposit: number, rate: number, threshold) => {
2  let amount = deposit
3  let year = 0
4
5  // while amount is lower than threshold
6  while (amount < threshold) {
7    // set amount equal to amount * the interest rate
8    amount = (amount * (1 + (rate / 100))).toFixed(2)
9    // increment the year
10    year += 1
11  }
12
13  // return the year amount is greater than threshold
14  return year
15}

#3 - Chunky Monkey

Problem

Given an array of values and a size, split the array into a 2-d array of length size.

Solution

1function chunkyMonkey(values, size) {
2    //  write code here.
3    let result = []
4    let id = 0
5
6    while (id < values.length) {
7        result.push(values.slice(id, id += size))
8    }
9
10    return result
11}

While id is less than the array length, push() a slice of the array onto the results array from id to id plus the desired size, while incrementing id by size for next go through while loop.

#4 - Century from Year

Problem

Given a year find the century.

Solution

1function centuryFromYear(year: number) {
2  return Math.ceil(year / 100)
3}

While the hint provided suggested to use Math.floor, I found that I could accomplished the desired result by using Math.ceil(). From seeing some of the other answers that use Math.floor() and where much more complex, I think this was the wrong hint to give.

#5 - Reverse a string

Problem

Given a string, reverse it.

Solution

1function reverseAString(str) {
2    //  write code here.
3    return str.split('').reverse().join('')
4}

Nice and simple. Hint was too revealing IMHO.

#6 - Sort by Length

Problem

Sort a given array of strings by their length.

Solution

1const sortByLength = (strs: string[]) => strs.sort((a, b) => a.length - b.length)

Using a simply comparer function for sort() based of the length I got the desired result.

#7 - Count Vowels and Consonants

Problem

Given a string, return the sum total of vowels and consonants, if vowels equal 1 pts. and consonants equal 2 pts.

Solution

1function countVowelConsonant(str: string) {
2  // write code here
3  const vowels = ['a', 'e', 'i', 'o', 'u']
4
5  return str.split('').reduce((acc, c) => {
6      if (vowels.includes(c)) {
7          return acc + 1
8      }
9      return acc + 2
10  }, 0)
11}

First I used a vowels array so that I can check if the current character is included in the array. Then I split the string into an array so I can reduce the list of characters down to a single value. When the current character c in included in the vowels array, add 1, if not add 2.
